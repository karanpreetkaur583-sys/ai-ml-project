#  Mood-Based Music Recommender

##  About the Project

This project is a simple Mood-Based Music Recommender built using basic Machine Learning concepts.
It takes user input (how the user is feeling), detects the mood, and suggests songs accordingly.

The main objective of this project is to apply machine learning concepts in a simple and practical way.

---

##  Objective

- To understand basic text classification
- To build a simple ML-based recommendation system
- To create a user-friendly music suggestion tool

---

##  How It Works

- The program is trained on a small dataset of sentences.
- It learns which words are associated with different moods like:
  - Happy
  - Sad
  - Angry
  - Relaxed
- When the user enters a sentence:
  - The program checks matching words
  - Predicts the mood

- Then it asks for music preference (pop/slow)
- Finally, it recommends songs

---

##  Technologies Used

- Python
- Basic Machine Learning logic (inspired by Naive Bayes)
